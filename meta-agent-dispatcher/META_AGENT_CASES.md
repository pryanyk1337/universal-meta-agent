# ПРИМЕРЫ КЕЙСОВ | Практика работы мета-агента

## КЕЙС 1: ЛИТЕРАТУРНЫЙ ПРОЕКТ

### Запрос пользователя
"Хочу написать научно-фантастический роман о колонии на Марсе. Нужна помощь с созданием всего проекта от начала до конца."

### Анализ мета-агента
```
Тип задачи: Комплексная литературная работа
Домен: Творчество → Литература → НФ
Сложность: Высокая (требует нескольких агентов)
```

### План выполнения
```
1. AGENT-001 (Аналитик-Архивариус):
   - Создать базу мира Марса
   - Разработать персонажей (5-7 главных)
   - Систематизировать технологии

2. AGENT-002 (Архитектор Сюжетов):
   - Разработать 3-актную структуру
   - Создать сюжетные арки персонажей
   - Определить ключевые конфликты

3. AGENT-003 (Писатель):
   - Написать роман по плану
   - Использовать базу мира
   - Соблюдать стилистику НФ
```

### Выполнение (последовательность)
**Шаг 1:** Мета-агент активирует AGENT-001
- Вход: "НФ роман, Марс, колония"
- Выход: База данных (15 стр.):
  * 7 персонажей с биографиями
  * 10 локаций колонии
  * 12 технологий будущего
  * Временная шкала событий

**Шаг 2:** Мета-агент передаёт данные в AGENT-002
- Вход: База данных от AGENT-001
- Выход: Сюжетный план:
  * Акт 1: Прибытие и адаптация (8 глав)
  * Акт 2: Кризис и конфликты (10 глав)
  * Акт 3: Разрешение (6 глав)
  * Дуги развития всех персонажей

**Шаг 3:** Мета-агент передаёт всё в AGENT-003
- Вход: База + Сюжет
- Выход: Роман (250-300 стр.)

### Результат
✅ Полный проект: база мира + сюжет + текст романа
✅ Согласованность всех элементов
✅ Выполнено за 3 этапа

"Помогите! Работодатель задерживает зарплату уже два месяца. Что мне делать?"

### Анализ мета-агента
```
Тип задачи: Консультация
Домен: Юриспруденция → Трудовое право
Срочность: Высокая
Сложность: Средняя (один агент)
```

### План выполнения
```

- Проанализировать ситуацию
- Предоставить правовую базу
- Дать пошаговую инструкцию
- Образцы документов
```

### Выполнение
**Одношаговое:**
- Вход: Описание ситуации
- AGENT-009 активирован
- Выход: Полная консультация:
  * Ваши права (ТК РФ ст. 136, 142)
  * Пошаговая инструкция
  * Образец письменной претензии
  * Контакты трудовой инспекции
  * Сроки и последствия

### Результат
✅ Быстрая помощь (один агент)
✅ Конкретные действия
✅ Правовая основа

## КЕЙС 3: БИЗНЕС-СТРАТЕГИЯ

### Запрос пользователя
"Планирую открыть кофейню в спальном районе. Нужен бизнес-план и маркетинговая стратегия."

### Анализ мета-агента
```
Тип задачи: Бизнес-планирование
Домен: Бизнес → Стратегия + Маркетинг
Сложность: Средняя (два агента)
```

### План выполнения
```
1. AGENT-006 (Бизнес-Стратег):
   - SWOT-анализ
   - Анализ конкурентов
   - Финансовая модель
   - Стратегический план

2. AGENT-007 (Маркетолог):
   - Целевая аудитория
   - Позиционирование
   - Маркетинг-план
   - Контент-стратегия
```

### Выполнение
**Шаг 1:** AGENT-006
- Вход: "Кофейня, спальный район"
- Выход: Стратегический анализ:
  * SWOT-анализ
  * 5 конкурентов изучены
  * Финансовая модель (3 года)
  * Уникальное предложение

**Шаг 2:** AGENT-007 (использует данные AGENT-006)
- Вход: Стратегия + Концепция
- Выход: Маркетинг-план:
  * Портрет ЦА (3 сегмента)
  * Позиционирование
  * 6 каналов продвижения
  * Контент-план на 3 месяца
  * Бюджет на рекламу

### Результат
✅ Комплексный бизнес-план
✅ Готовая маркетинговая стратегия
✅ Можно начинать реализацию

## КЕЙС 6: КОМБИНИРОВАННАЯ ЗАДАЧА

### Запрос пользователя
"Напиши стихотворение о весне и создай под него описание для публикации в соцсетях"

### Анализ мета-агента
```
Тип задачи: Комбинированная
Домен 1: Творчество → Поэзия
Домен 2: Бизнес → Маркетинг
Сложность: Средняя (два агента)
```

### План выполнения
```
1. AGENT-004 (Поэт):
   - Создать стихотворение о весне
   
2. AGENT-007 (Маркетолог):
   - Написать пост для соцсетей
   - Использовать стихотворение
```

### Выполнение
**Шаг 1:** AGENT-004
- Вход: "Тема: весна, форма: любая"
- Выход: Стихотворение (12 строк, классическая рифмовка)

**Шаг 2:** AGENT-007
- Вход: Стихотворение + "Пост для соцсетей"
- Выход: 
  * Продающий текст-вступление
  * Стихотворение (встроено)
  * Призыв к действию (лайк, репост)
  * Хештеги (#весна #поэзия #творчество)

### Результат
✅ Стихотворение создано
✅ Готовый пост для публикации
✅ Два агента работали последовательно

## КЕЙС 7: ВОССТАНОВЛЕНИЕ КОНТЕКСТА

### Ситуация
Пользователь продолжает работу из предыдущего чата, который был удалён и восстановлен.

### Запрос пользователя
"Продолжай работу"

### НЕПРАВИЛЬНОЕ поведение мета-агента
❌ "Хорошо, продолжаю" (без понимания задачи)
❌ Начинает действовать наугад
❌ Ложная уверенность

### ПРАВИЛЬНОЕ поведение мета-агента
✅ "Уточните, пожалуйста, какую работу продолжить:
- О каком проекте речь?
- На каком этапе остановились?
- Что уже сделано?
- Что осталось сделать?

Это поможет мне продолжить работу точно."

### Результат
✅ Избежание ошибки
✅ Восстановление полного контекста
✅ Правильное продолжение работы

"Помоги написать медицинское заключение по результатам анализов"

### Анализ мета-агента
```
Тип задачи: Медицинская консультация
Домен: НЕТ В РЕЕСТРЕ
Компетенция: ОТСУТСТВУЕТ
```

```
Мета-агент: "К сожалению, у меня нет агентов с медицинской специализацией.

Медицинские заключения должны составляться только квалифицированными врачами.

Рекомендую обратиться:
1. К вашему лечащему врачу
2. В медицинское учреждение
3. К специалисту в вашей области

Могу помочь с другими задачами:
- Литература, творчество
- Бизнес-планирование
- Программирование
- Юридические консультации (не медицинские)"
```

### Результат
✅ Честное признание ограничений
✅ Направление к правильным источникам
✅ Предложение альтернативной помощи
