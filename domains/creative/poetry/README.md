# Домен: Поэзия

## Описание области
Поэзия — особый вид литературного творчества, использующий ритм, рифму, метафоричность и образность для передачи эмоций и идей в концентрированной форме.

## Ключевые компетенции
- Работа с ритмом и метром
- Рифмовка
- Создание образов и метафор
- Звукопись и аллитерация
- Эмоциональная выразительность
- Понимание поэтических форм

## Стихотворные размеры
### Двусложные:
- **Ямб**: безударный-ударный (˘ /)
- **Хорей**: ударный-безударный (/ ˘)

### Трёхсложные:
- **Дактиль**: ударный-безударный-безударный (/ ˘ ˘)
- **Амфибрахий**: безударный-ударный-безударный (˘ / ˘)
- **Анапест**: безударный-безударный-ударный (˘ ˘ /)

### Современные формы:
- **Верлибр** (свободный стих): без рифмы и размера
- **Белый стих**: есть размер, но нет рифмы

## Типы рифм
### По положению в строке:
- **Смежная** (AABB): рифмуются соседние строки
- **Перекрёстная** (ABAB): чередующаяся рифма
- **Кольцевая** (ABBA): опоясывающая рифма

### По точности:
- **Точная**: полное созвучие (мечта — красота)
- **Неточная**: частичное созвучие (мечта — весна)
- **Богатая**: рифмуются несколько звуков
- **Бедная**: рифмуется только ударный слог

## Поэтические формы
### Классические:
- **Сонет**: 14 строк с определённой схемой рифмовки
- **Терцина**: трёхстрочные строфы (ABA BCB CDC)
- **Октава**: 8-строчная строфа (ABABABCC)
- **Рондо**: со строками-рефренами

### Восточные:
- **Хайку**: трёхстишие 5-7-5 слогов
- **Танка**: пятистишие 5-7-5-7-7 слогов
- **Газель**: парные двустишия

### Экспериментальные:
- **Визуальная поэзия**: форма текста создаёт образ
- **Акростих**: первые буквы строк образуют слово
- **Палиндром**: читается одинаково в обе стороны

## Художественные средства
### Тропы (образные средства):
- **Метафора**: скрытое сравнение (жизнь — река)
- **Олицетворение**: неживое наделяется жизнью
- **Эпитет**: образное определение (золотая осень)
- **Гипербола**: преувеличение
- **Литота**: преуменьшение

### Звуковые приёмы:
- **Аллитерация**: повтор согласных звуков
- **Ассонанс**: повтор гласных звуков
- **Анафора**: повтор начала строк
- **Эпифора**: повтор конца строк

## Тематические направления
- **Лирика любви**: чувства, отношения
- **Пейзажная лирика**: природа, времена года
- **Философская лирика**: размышления о жизни
- **Гражданская лирика**: социальные темы
- **Религиозная поэзия**: духовные темы
- **Ироническая поэзия**: юмор, сатира

## Процесс создания стихов
1. **Вдохновение**: эмоция, образ, идея
2. **Выбор формы**: размер, рифма, структура
3. **Черновик**: первая версия
4. **Работа со звучанием**: ритм, рифмы
5. **Шлифовка образов**: метафоры, эпитеты
6. **Редактура**: устранение лишнего
7. **Финальная вычитка**: проверка на слух

## Советы поэтам
1. Читайте стихи вслух — слушайте звучание
2. Избегайте банальных рифм (любовь-кровь)
3. Каждое слово должно быть на своём месте
4. Не жертвуйте смыслом ради рифмы
5. Экспериментируйте с формами
6. Изучайте классиков и современников
7. Пишите регулярно

## Применение в агентах
Агенты-поэты могут:
- Создавать стихотворения на заказ
- Обучать стихосложению
- Анализировать поэтические произведения
- Помогать с редактированием стихов
- Адаптировать стиль под задачу
- Работать с различными формами и жанрами
